## Installation

***Please backup any existing configuration files***

### Clone the repository

#### Linux

```text
git clone https://github.com/AkagiYuu/neovim-config ~/.config/nvim
nvim +PackerSync
```

#### Windows

```text
git clone https://github.com/AkagiYuu/neovim-config ~/AppData/Local/nvim
nvim +PackerSync
```
